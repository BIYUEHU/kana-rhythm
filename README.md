## Environment

- `Zed`
- `MoonBit`
- `pnpm`
- `Node.js`
- `Python`

## Development

```bash
pnpm watch # Compile MoonBit and watch for changes
pnpm dev # Start development server base on Vite
```

## Build

```bash
pnpm build # Compile MoonBit and build web app
```
